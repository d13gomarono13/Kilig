# Kilig 2.0

AI platform to transform scientific papers into animated videos using a Data-Driven Pure TypeScript approach.

## Running the Agent

This project leverages [tsx](https://tsx.is) to execute the TypeScript agent.

```sh
pnpm dev
```

## Operations

**Checking Types**

```sh
pnpm check
```

**Testing**

```sh
pnpm test
```
