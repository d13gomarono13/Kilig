# Theme Factory

Apply professional styling to presentations and artifacts.

## Usage

1. Show `theme-showcase.pdf` to display all themes
2. Ask user to choose a theme
3. Apply selected theme's colors and fonts

## Available Themes

1. **Ocean Depths** - Professional maritime
2. **Sunset Boulevard** - Warm sunset colors
3. **Forest Canopy** - Natural earth tones
4. **Modern Minimalist** - Clean grayscale
5. **Golden Hour** - Rich autumnal palette
6. **Arctic Frost** - Cool winter-inspired
7. **Desert Rose** - Soft dusty tones
8. **Tech Innovation** - Bold tech aesthetic
9. **Botanical Garden** - Fresh organic colors
10. **Midnight Galaxy** - Dramatic cosmic tones

## Theme Structure

Each theme in `themes/` directory includes:
- Cohesive color palette with hex codes
- Complementary font pairings (headers + body)
- Visual identity guidelines

## Custom Themes

For requests not matching existing themes:
1. Generate custom theme based on input
2. Choose appropriate colors/fonts
3. Give descriptive name
4. Show for review before applying
